### **HTML Requirements**

* Must include 

  * Sections with headings

    * More than one level of headings

  * Hyperlinks

  * Images

    * At least one non-decorative (i.e. the image contains meaningful content)

    * At least one decorative

  * Form to collect information

    * Forms must initiate an action.

      * For example, [forms can be created that send an email](https://www.w3schools.com/html/tryit.asp?filename=tryhtml_form_mail). 

        * Using the email approach is fine if you don't know how to use scripting to collect data.

  * Lists

    * At least one list with two items.

  * Semantic text emphasis

    * Use both strong and em.

  * (Optional) non-formatting table

  * Correctly link to external CSS

* Cannot have

  * Any internal or inline CSS

  * Any older formatting html that should be done with CSS (e.g. color)

  * Any non-semantic text markup/emphasis (e.g. bold, italics)

* Must pass [W3C Markup Validation](https://validator.w3.org/detailed.html).

### **CSS Requirements**

* Must be an external stylesheet.

* Must update at least one of the properties for each of the following styling properties.

  * Typography

    * E.g. Font family, size, weight, and color

  * Layout

    * E.g. Margins, padding, borders, positioning

  * Feel free to do additional updates that improve the look of your page while keeping it accessible.

* Must pass [W3C CSS Validation](https://jigsaw.w3.org/css-validator/).
